# Install

In terminal,
```bash
$>_ npm i --save f.ck
```

# Usage

```javascript
var f = require('f.ck');

f('++++++++++[>+++++++>++++++++++>+++>+<<<<-]>++.>+.+++++++..+++.>++.<<+++++++++++++++.>.+++.------.--------.>+.>.')
  .then(value => console.log(value))
  .catch(err => console.log(err))
```
